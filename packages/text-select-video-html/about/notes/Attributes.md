# Attributes


Attributes for the `<video-text-select>` tag:

    autoplay : boolean
    controls : boolean
    height : string | number
    loop : boolean
    muted : boolean
    poster : string | filepath | url
    pregenerate : boolean
    preload : string
    src (or source) : string | filepath | url
    sources : string[] | filepath[] | url[]
    width : string | number
